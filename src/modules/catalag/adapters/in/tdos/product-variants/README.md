# Product Variant TDOS

DTOs HTTP de variantes de producto.

## Archivos

- `http-variant-create.dto.ts`: payload de creacion.
- `http-variant-update.dto.ts`: payload de actualizacion.
- `http-variant-set-active.dto.ts`: payload de activar/desactivar.
- `http-variant-list.dto.ts`: query de listado y busqueda.
