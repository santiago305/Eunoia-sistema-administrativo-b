# Product Inputs

DTOs de entrada para productos.

## Archivos

- `create-product.ts`: input de creacion.
- `update-product.ts`: input de actualizacion.
- `set-active-product.ts`: input de activar/desactivar.
- `get-by-id-product.ts`: input de consulta por id.
- `list-products.ts`: input de busqueda y paginacion.
