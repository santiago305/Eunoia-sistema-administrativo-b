# Product Variant Inputs

DTOs de entrada para variantes.

## Archivos

- `create-product-variant.ts`: input de creacion.
- `update-product-variant.ts`: input de actualizacion.
- `set-active-product-variant.ts`: input de activar/desactivar.
- `get-by-id-product-variant.ts`: input de consulta por id.
- `list-product-variant.ts`: input de busqueda y listado.
- `inputs.ts`: exportaciones de inputs.
