# Product Use Cases

Casos de uso de productos.

## Archivos

- `created.usecase.ts`: crea un producto.
- `created.usecase.spec.ts`: pruebas unitarias de creacion.
- `update.usecase.ts`: actualiza un producto.
- `update.usecase.spec.ts`: pruebas unitarias de actualizacion.
- `set-active.usecase.ts`: activa o desactiva un producto.
- `set-active.usecase.spec.ts`: pruebas unitarias de activar/desactivar.
- `list-active.usecase.ts`: lista productos activos.
- `list-active.usecase.spec.ts`: pruebas unitarias de listado activo.
- `list-inactive.usecase.ts`: lista productos inactivos.
- `list-inactive.usecase.spec.ts`: pruebas unitarias de listado inactivo.
- `search-paginated.usecase.ts`: busca productos con paginacion.
- `search-paginated.usecase.spec.ts`: pruebas unitarias de busqueda paginada.
- `get-with-variants.usecase.ts`: obtiene producto con variantes.
- `get-with-variants.usecase.spec.ts`: pruebas unitarias de consulta con variantes.
