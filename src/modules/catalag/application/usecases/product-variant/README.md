# Product Variant Use Cases

Casos de uso de variantes de producto.

## Archivos

- `create.usecase.ts`: crea una variante.
- `create.usecase.spec.ts`: pruebas unitarias de creacion.
- `update.usecase.ts`: actualiza una variante.
- `update.usecase.spec.ts`: pruebas unitarias de actualizacion.
- `set-active.usecase.ts`: activa o desactiva una variante.
- `set-active.usecase.spec.ts`: pruebas unitarias de activar/desactivar.
- `get-element-by-id.usercase.ts`: obtiene variante por id.
- `get-element-by-id.usercase.spec.ts`: pruebas unitarias de consulta por id.
- `list-by-product.usecase.ts`: lista variantes por producto.
- `list-by-product.usecase.spec.ts`: pruebas unitarias de listado por producto.
- `list-active.usecase.ts`: lista variantes activas por producto.
- `list-active.usecase.spec.ts`: pruebas unitarias de listado activo.
- `list-inactive.usecase.ts`: lista variantes inactivas por producto.
- `list-inactive.usecase.spec.ts`: pruebas unitarias de listado inactivo.
- `search.usecase.ts`: busca variantes por filtros.
- `search.usecase.spec.ts`: pruebas unitarias de busqueda.
